Event Management API

GET - /events

Get a list of all events

GET - /events/:id

Get an event by id

GET - /events/by-name/:name

Get an event by name

POST - /events

Create an event

Pass the event information as a JSON object in the request body. You do not need to provide an id, as the db will automatically generate one.

PUT - /events/:id

Update an event by id

DELETE - /events/:id

Delete an event by id

DELETE - /events

Delete all events
